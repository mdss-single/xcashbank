Атрибуты

data-tooltip="%text%" Добавляет тултип для элемента. Работает только на десктоп версии
data-tooltip-left="%text%" Зеркальное отображение тултипа

Модальные окна

Для вызова модальных окон подключен плагин fancybox. Для вызывающего элемента необходимо добавить класс .js-modal. Если это ссылка, то вызываемый адрес пишет в атрибуте href, если какой-то другой элемент, то прописывается атрибут data-src="%target%"
